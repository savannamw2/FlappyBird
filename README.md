## Overview

**Project Title**: Bobs Burgers FlappyBird

**Project Description**: Use the space bar to move the plain around the obstacles. 

**Project Goals**: Calculate user score and move around obstacles as much as possible. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download Pygame
2. import sys and time and pygame
3. create classes
4. Create methods

Instructions for using the software:

1. Download globally to environment 
2. Import libraries
3. Compile and Run Game

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Python
* Pygame 
* sys
* time
